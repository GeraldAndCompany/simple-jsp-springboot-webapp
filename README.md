# simple-jsp-springboot-webapp

Requires:
 - Java 8
 - Maven 3
 
To run:
  - in the project directory
  - $ mvn clean install
  - $ mvn tomcat7:install
  
 To access webapp:
  - access http://localhost:9090/app
  
 - https is enabled by using a self-signed cert. It can be accessed by going to https://localhost:8443/app
